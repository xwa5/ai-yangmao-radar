# 需求文档

## 1. 应用概述

- **应用名称**：AI 羊毛雷达
- **应用描述**：一款微信小程序，自动聚合各大 AI 平台的最新优惠活动（免费额度、试用资格、积分奖励、API 赠送、红包折扣等），提供活动聚合浏览与 Agent 智能问答两种使用方式，帮助用户高效获取各类 AI 资源福利。

---

## 2. 用户与使用场景

### 2.1 目标用户

- 关注 AI 平台优惠活动、希望低成本使用 AI 服务的个人用户
- 开发者、AI 爱好者等需要 API 额度或试用资格的用户

### 2.2 核心使用场景

- 用户打开小程序，浏览当前最新的 AI 平台薅羊毛活动列表
- 用户通过 Agent 对话，输入如「查询 GPT 优惠活动」，获取针对性活动信息
- 用户登录后，查看自己的历史查询记录与收藏活动

---

## 3. 页面结构与核心功能

### 3.1 页面总览

```
小程序
├── 首页（活动聚合页）
├── Agent 对话页
├── 活动详情页
├── 个人中心页
│   ├── 登录/注册
│   ├── 历史记录
│   └── 收藏列表
```

### 3.2 首页（活动聚合页）

**功能说明：**

- 展示活动卡片列表，数据来源为后端数据库（定时爬取入库），初期展示示例数据
- 每张活动卡片包含以下字段：
  - 平台名称（如 OpenAI、Claude、Gemini 等）
  - 活动标题
  - 活动类型标签（免费额度 / 试用资格 / 积分奖励 / API 赠送 / 红包折扣 / 限时福利）
  - 适用人群（新用户 / 所有用户 / 开发者等）
  - 活动截止时间（若有）
  - 简短描述
- 支持按活动类型标签筛选
- 支持按平台名称筛选
- 点击活动卡片跳转至活动详情页
- 顶部提供入口跳转至 Agent 对话页

### 3.3 Agent 对话页

**功能说明：**

- 提供对话输入框，用户输入自然语言需求（如「查询 GPT 优惠活动」「有哪些免费 API 可以薅」）
- Agent 基于数据库中的活动数据进行检索与整理，返回结构化活动清单
- 返回内容包含：活动名称、参与方式、适用人群、截止时间
- 支持多轮对话，用户可追问细节
- 登录用户的对话记录自动保存至历史记录
- 未登录用户可正常使用对话功能，但不保存历史记录

### 3.4 活动详情页

**功能说明：**

- 展示活动完整信息：
  - 平台名称与 Logo（示例数据阶段可用占位图）
  - 活动标题与类型
  - 详细描述
  - 参与方式（步骤说明）
  - 适用人群
  - 活动有效期
  - 原始活动链接（可跳转外部浏览器）
- 登录用户可收藏该活动
- 提供「分享」按钮，支持微信内转发

### 3.5 个人中心页

**功能说明：**

- **未登录状态**：展示登录引导，提示登录后可使用历史记录与收藏功能
- **登录/注册**：支持微信授权登录 + 账号密码登录
- **已登录状态**：
  - 展示用户昵称与头像（微信授权登录时自动获取）
  - 历史记录：按时间倒序展示用户在 Agent 对话页的历史对话列表，点击可查看对话详情
  - 收藏列表：展示用户收藏的活动卡片，支持取消收藏
  - 退出登录

---

## 4. 业务规则与逻辑

- 活动数据由后端定时爬取并存入数据库，小程序端直接读取数据库数据；初期使用示例数据填充展示
- Agent 对话的数据检索范围限定为当前数据库中已有的活动数据，不进行实时网络搜索
- 收藏与历史记录功能仅对已登录用户开放；未登录用户使用 Agent 对话时，对话内容不持久化
- 活动卡片按入库时间倒序排列，最新活动优先展示
- 活动截止时间过期后，对应活动卡片在列表中标记为「已结束」，仍可查看详情但不可收藏

---

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 数据库暂无数据 | 首页展示示例数据占位，提示「数据更新中，以下为示例展示」 |
| Agent 未匹配到相关活动 | 返回提示「暂未找到相关活动，可尝试换个关键词」 |
| 用户未登录时点击收藏 | 弹出登录引导弹窗 |
| 活动原始链接失效 | 详情页提示「原始链接暂时无法访问，请稍后再试」 |
| 网络请求失败 | 展示通用错误提示，提供重试按钮 |

---

## 6. 验收标准

- 首页能正常展示活动卡片列表，支持按类型和平台筛选，示例数据完整呈现
- Agent 对话页能接收用户输入并返回结构化活动信息，支持多轮对话
- 活动详情页信息完整，外部链接可正常跳转，分享功能可用
- 微信授权登录与账号密码登录均可正常完成
- 登录用户的历史记录与收藏数据可正常读写，退出登录后数据不丢失
- 未登录用户可浏览首页与使用 Agent 对话，但收藏与历史记录入口提示需登录
- 已结束活动在列表中正确标记状态

---

## 7. 本期不实现功能

- 活动数据的自动爬取与入库逻辑（本期仅使用示例数据，爬取能力由后端独立实现）
- 活动数据的人工审核与编辑后台
- 用户评论、评分或反馈功能
- 消息推送（新活动上线通知）
- 活动参与进度追踪
- 多语言支持
